# Formação - Aprofunde em Java com Arquitetura de Microsserviços, Spring e RabbitMQ

### Esse projeto foi desenvolvido durante os cursos dessa formação, onde aprendemos a criar uma aplicação de microsserviços usando Java, Spring Boot e RabbitMQ. O projeto inclui a implementação de um serviço de pedidos que se comunica com outros serviços através do RabbitMQ, além de utilizar o AWS CDK para provisionar a infraestrutura necessária na AWS. Outra forma de utilizar também é utilizando o Eureka Server para registro e descoberta de serviços, e o Spring Cloud Gateway para roteamento de requisições.

Foram desenvolvidos os seguintes projetos:

- [Microsserviço de Pedidos](https://github.com/matheus-vsm/AluraAluraFoodPedidosMicrosservicosJavaSpringCourse);

- [Microsserviço de Pagamentos](https://github.com/matheus-vsm/AluraAluraFoodPagamentosMicrosservicosJavaSpringCourse);

- [Microsserviço de Avaliação](https://github.com/matheus-vsm/AluraAluraFoodAvaliacaoMicrosservicosJavaSpringCourse);

- [Serviço da AWS](https://github.com/matheus-vsm/AluraAluraFoodAWSMicrosservicosJavaSpringCourse);

- [Serviço do Servidor Eureka](https://github.com/matheus-vsm/AluraAluraFoodServerMicrosservicosJavaSpringCourse);

- [Microsserviço do Gateway](https://github.com/matheus-vsm/AluraAluraFoodGatewayMicrosservicosJavaSpringCourse).
